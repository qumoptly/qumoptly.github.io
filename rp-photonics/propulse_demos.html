Demonstrations for Models Made with RP ProPulse

Various example cases illustrate what you can do with RP ProPulse:

  • active mode locking
  • passive mode locking
  • soliton mode-locked bulk laser
  • mode-locked fiber laser
  • higher-order solitons
  • nonlinear pulse compression
  • adiabatic soliton compression
  • soliton self-frequency shift

(Click on the links to get the details.)

The input scripts for these and some other cases are available to license holders. For an easy start in a new project, you can simply copy the example file of a case which is closest to the case of interest. However, you may also easily generate a new script using the interactive forms.
