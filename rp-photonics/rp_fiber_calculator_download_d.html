Download und Installation

Diese Software funktioniert auf Computern, die unter Microsoft Windows laufen. (Eine beliebige Windows-Version ab XP kann benutzt werden, nur nicht in einer virtuellen Maschine.)

Laden Sie einfach das Installationsprogramm herunter:

Download der kostenlosen Software RP Fiber Calculator

Sie können dann die Software ganz einfach installieren, indem Sie das Installationsprogramm ausführen.

Wenn Sie Windows 8 oder 10 benutzen (oder Internet Explorer auf anderen Systemen), erhalten Sie u. U. eine Meldung wie “Windows SmartScreen prevented an unrecognized app from starting ...”. In diesem Fall können Sie auf “More info” gehen, den “Publisher” verifizieren (RP Photonics Consulting GmbH) und “Run anyway” wählen – aber nur, wenn Sie sicher sind, dass Sie das Programm von unserer Website heruntergeladen haben, also das Original.

Sollten Sie irgendein Problem mit dem Download oder der Installation haben, wären wir für einen Hinweis z. B. via E-Mail dankbar. Dies gilt auch für Nutzer der kostenlosen Version.

Wir versichern Ihnen, dass wir jedenfalls nicht absichtlich oder wissentlich unsere Software mit irgendwelchen schädlichen Funktionen ausgestattet haben, etwa um etwas auf Ihrem PC zu zerstören, ihn mit Schadcode zu infizieren oder Ihre Daten zu stehlen. Aber natürlich haben wir keine Kontrolle darüber, ob von Dritten modifizierte Versionen illegal auf anderen Websites zum Download angeboten werden. Und wir können keine Haftung übernehmen für böswillige Angriffe von Dritten, falls solche trotz großer Sorgfalt von unserer Seite nicht unterbunden werden können.

Lizenzbedingungen

Die Installation der Software setzt voraus, dass Sie unsere Lizenzbedingungen akzeptieren. Diese sind sehr großzügig. Die wichtigsten Punkte:

Sie können diese Software anonym nutzen, wenn Sie wollen. Sie müssen uns keinerlei Daten geben, um die Software zu nutzen.
  • Sie dürfen die Software kostenlos benutzen – sogar für die Arbeit an kommerziellen Projekten.
  • Wir verlangen von Ihnen nicht, dass Sie sich bei uns registrieren.
  • Sie dürfen die Software nicht an andere Personen oder Institutionen weitergeben. Jeder Benutzer soll sie selbst auf unserer Website herunterladen. Damit stellen wir sicher, dass alle Benutzer die Original-Software bekommen (nicht etwa eine, die von Dritten mit Schadsoftware infiziert wurde!) und die jeweils neueste Version installieren.
  • Wir respektieren den Datenschutz uneingeschränkt. Unsere Software wird lediglich ab und zu über unseren Server prüfen, ob ein Update oder aktuelle Hinweise verfügbar sind, und erlaubt Ihnen weitere Informationen über unsere Website abzurufen. Wir versuchen nicht, die Benutzer unserer kostenlosen Software zu identifizieren.

Ihre Rückmeldungen sind stets willkommen – beispielsweise wenn Sie Fehler gefunden haben oder Verbesserungen vorschlagen möchten. Bitte verstehen Sie aber, dass wir für kostenlose Software nicht noch weiteren technischen Support bieten können. Selbstverständlich können Sie bezahlte technische Beratung von RP Photonics erhalten.
