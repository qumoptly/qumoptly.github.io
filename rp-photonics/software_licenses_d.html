Benutzerlizenzen für Modellierungs-Software

Kommerzielle Lizenzen

Lizenzen für die kommerzielle Benutzung von RP Resonator, z. B. durch Firmen, die Laser herstellen und entwickeln, können von RP Photonics erworben werden.

Die Lizenzgebühr beinhaltet 8 Stunden technischen Support (außer bei RP Fiber Calculator PRO).

Nichtkommerzielle Lizenzen

Universitäten und andere Ausbildungsstätten können Lizenzen für nichtkommerzielle Benutzung zu einem erheblich reduzierten Preis erwerben, wenn die Software nur für öffentliche Forschung (d. h. öffentlich finanzierte Forschung, deren Resultate normalerweise publiziert werden) und für Ausbildungszwecke (z. B. in Kursen über Lasertechnologie oder Faseroptik) genutzt wird. Es ist nicht erlaubt, solche Lizenzen für die Zusammenarbeit mit einem Partner zu verwenden, der kommerzielle Interessen verfolgt, außer dieser Partner besitzt selbst eine kommerzielle Lizenz.

Die Lizenzgebühr beinhaltet 4 Stunden technischen Support (außer bei RP Fiber Calculator PRO).

Permanente oder zeitliche begrenzte Lizenzen

Normalerweise verkauft RP Photonics permanente (zeitlich unbeschränkte) Lizenzen; Sie zahlen einmalig die Lizenzgebühr und können die Software dann für immer verwenden. Jedoch bieten wir seit 2015 auch zeitlich beschränkte Lizenzen an. In diesem Fall sind die Regeln wie folgt:

Neu: zeitlich begrenzte Softwarelizenzen erhältlich! Benutzen Sie solche für kurzzeitige Projekte oder für intensive Tests.
  • Die Dauer der Lizenz kann eine beliebige Anzahl von Monaten betragen. Die Lizenzgebühr pro Monat ist 5 % des Betrags für eine permanente Lizenz.
  • Die eingeschlossene Support-Zeit ist eine halbe Stunde pro Monat bei kommerziellen Lizenzen und eine Viertelstunde pro Monat bei nichtkommerziellen Lizenzen. (Ausnahme: Für RP Fiber Calculator PRO gibt es nur halb so viel Supportzeit.) Natürlich können Sie auch die gesamte Supportzeit schon im ersten Monat aufbrauchen, wenn Sie die Nutzung für mehrere Monate bestellen. Im übrigen ist es jederzeit möglich, zusätzliche Support-Stunden zu kaufen.
  • Wenn Sie sich innerhalb der ersten drei Monaten der Lizenzperiode für den Übergang zu einer permanenten Lizenz entscheiden, wird der bereits bezahlte Betrag von der Lizenzgebühr für die permanente Lizenz komplett abgezogen. Wenn Sie diese Entscheidung später vornehmen, wird der bereits bezahlte Betrag zu zwei Dritteln angerechnet.

Wenn Sie die Software umfassend testen möchten, bevor Sie definitiv damit arbeiten, können Sie einfach eine Lizenz für ein oder zwei Monate kaufen. Wir gehen davon aus, dass Sie vollkommen überzeugt sein werden und dann die permanente Lizenz erwerben, ohne für die Testphase zusätzliches Geld ausgegeben zu haben.

Updates und Upgrades

Software von RP Photonics wird ständig weiter entwickelt: Details werden verbessert, Fehler werden behoben, etc. Besitzer von Nutzerlizenzen haben ein Recht auf kostenlose Updates der Version, die sie jeweils gekauft haben – selbst Jahre nach dem Kauf. Andere haben dafür regelmäßig teuer zu bezahlende Wartungsverträge – wir nicht.

Sie erhalten kostenlose Updates und bezahlen nur für Upgrades auf stark verbesserte Versionen!

Wenn eine grundlegend neue Version erscheint, deren Lizenzen teurer sind als die der alten Version, erhalten die Kunden ein faires Upgrade-Angebot. Ein Upgrade kostet nur wenig mehr als die Differenz des neuen und alten Preises der Lizenz.

Wenn ein Kunde ein Upgrade-Angebot ausschlägt, z. B. weil er die neuen Features nicht benötigt, kann er trotzdem Updates auf die alte Version erhalten. Diese sehen dann wie die neue Version aus: Die neuen Features werden in der Dokumentation beschrieben, und die Benutzeroberfläche kann neue Elemente enthalten, nur dass der Code für die neuen Features nicht da ist.

Kunden werden regelmäßig über mögliche Upgrades und Updates (sowie andere interessante Dinge) informiert, wenn sie den Newsletter namens RP Photonics Software news abonnieren.

Technischer Support

Wenn Sie eine Software für komplexe Aufgaben einsetzen, ist ein kompetenter und hilfreicher technischer Support essenziell. RP Photonics nimmt dies sehr ernst. Details lesen Sie auf der Seite über den technischen Support.

Demo-Versionen

Wir werden gelegentlich nach kostenlosen Demo-Versionen gefragt, aber solche bieten wir nicht an. Natürlich verstehen wir, dass Sie vor einer Kaufentscheidung genau wissen möchten, was das Produkt für Sie leisten kann. Deswegen haben wir diverse Möglichkeiten geschaffen:

Es gibt eine Menge von Möglichkeiten, um die Details unserer Software genau zu erfahren.
  • Es gibt ausführliche Beschreibungen auf unserer Website. Dazu gehören auch diverse Videos und unzählige Fallstudien; betrachten Sie z. B. die letzteren für RP Fiber Power.
  • Gerne beantworten wir Ihre technischen Fragen (etwa per E-Mail oder über das Formular für Software-Anfragen) zur Software detailliert.
  • Wir bieten ernsthaften Interessenten auch eine Online-Demo an, bei der Sie per Internet Zugriff auf unseren Bildschirm erhalten, während wir uns unterhalten.
  • Wenn Sie die Software wirklich selbst ausgiebig testen möchten, können Sie dafür zu sehr moderaten Kosten eine zeitlich begrenzte Lizenz erwerben (siehe oben).

Eine weitere Option: Beratung

Sie müssen nicht unbedingt eine Lizenz besitzen, um von der Software profitieren zu können. Stattdessen (oder in Ergänzung) können Sie auch Beratung von Dr. Paschotta erhalten. Er kann einiges mehr für Sie tun, als irgendeine Software es könnte: die Analyse Ihrer Daten, Vorschläge für die Ausarbeitung optimierter Designs, die Erklärung technischer Limitationen, alternative technische Ansätze, etc. Und natürlich kommen Sie so noch schneller zu Ihren Resultaten.

Selbstverständlich können Sie später jederzeit doch noch eine Lizenz erwerben. Sie erhalten dann alle Skriptdateien, die RP Photonics bei der Arbeit an Ihrem Projekt ausgearbeitet hat, und können damit das Modell weiter ausbauen oder mit veränderten Parametern verwenden.
